Rule 4.12: "Neither"-VP Aggregation
===================================


```{rewrite-rule}
{ph}`VP_1`, $\ldots$, {ph}`VP_n` : {cat}`VP` = {nl}`contain $y$`, {nl}`include $z$`

{ph}`t` : {cat}`Term` = {nl}`$x$`

{ph}`t` does not {ph}`VP_1` and ... and {ph}`t` does not {ph}`VP_n`

{ph}`t` does neither {ph}`VP_1` nor ... nor {ph}`VP_n`
```

Examples
--------

```{rewrite-rule}
$x$ does not contain $y$ and $x$ does not include $z$

$x$ does neither contain $y$ nor include $z$
```
