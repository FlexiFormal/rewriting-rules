Rule 10.8: Modal "False"-AP Reduction
=====================================


```{rewrite-rule}
{ph}`AP` : {cat}`AP` = {nl}`prime`

{ph}`x` : {cat}`Term` = {nl}`$x$`

it (is wrong | is false | is not the case | does not hold) that {ph}`x` is {ph}`AP`

{ph}`x` is not {ph}`AP`
```


Examples
--------

```{rewrite-rule}
it does not hold that $n$ is prime

$n$ is not prime
```
