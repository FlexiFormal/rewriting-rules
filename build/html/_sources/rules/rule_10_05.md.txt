Rule 10.5: Modal "False"-VP Reduction
=====================================


```{rewrite-rule}
{ph}`VP` : {cat}`VP` = {nl}`intersect with $y$`

{ph}`x` : {cat}`Term` = {nl}`$x$`

it (is wrong | is false | is not the case | does not hold) that {ph}`x` {ph}`VP`

{ph}`x` does not {ph}`VP`
```


Examples
--------

```{rewrite-rule}
it does not hold that $x$ intersects with $y$

$x$ does not intersect with $y$
```
