Rule 11.5: VP Double Negation Elimination
=========================================


```{rewrite-rule}
{ph}`VP` : {cat}`VP` = {nl}`intersect with $y$`

{ph}`P` : {cat}`Var` = {nl}`$P$`

{ph}`x` : {cat}`Term` = {nl}`$x$`

$\vdash \forall {ph}`P`.\, \neg \neg {ph}`P` = {ph}`P`$

{ph}`x` does not not {ph}`VP`

{ph}`x` {ph}`VP`
```


Examples
--------

```{rewrite-rule}
$\vdash \forall P.\, \neg \neg P = P$

$x$ does not not intersect with $y$

$x$ intersects with $y$
```
