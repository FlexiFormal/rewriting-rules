Rule 11.3: Adjective Double Negation Elimination
================================================


```{rewrite-rule}
adjective : <adjective> = "empty"

P : <Var> = "$P$"

x : <Term> = "$x$"

|- $\forall :P:.\, \neg \neg :P: = :P:$

":x: is not non-:adjective:"

--- Adjective Double Negation Elimination

":x: is :adjective:"
```


Examples
--------

```{rewrite-rule}
|- $\forall P.\, \neg \neg P = P$

"$x$ is not non-empty"

"$x$ is empty"
```
