A guarded identifier is a formula that grammatically acts as an identifier but contains restrictions on the domain.

Example:

For every \(x \in \mathbb{R}\) ...

This can be treated as a short-form for:

For every \(x\) such that \(x \in \mathbb{R}\) ...

We typically write this as \(R[x]\), where \(R\) is the formula and \(x\) is the relevant identifier.

Some rewriting rules use guarded identifiers for simplification/contraction. Identifying if a formula can act as a guarded identifier is not trivial. For example, \(k < x < n\) can be a guarded identifier if \(k\) and \(n\) are not free.
