Rule 5.8: “Choose”-AP Comprehension Term Reduction¶

choose : <choose> = "choose"
AP : <AP> = "imaginary"
x : <Var> = "x"
y : <Var> = "z"
P[a] : <Pred> = "an=1 for some n∈ℕ"
choose a x∈{y∣P[y]} that is a AP
choose a AP x with P[x]

Examples¶

choose an x∈{z∣zn=1 for some n∈ℕ} that is imaginary
choose an imaginary x with xn=1 for some n∈ℕ
