Rule 6.4: Parametrized Iota Conversion¶

f : <Term> = "δ"
A : <Term> = "Q×Σ"
B : <Term> = "Q"
a : <Term> = "(q,c)"
b : <Term> = "q′"
⊢ f:A→𝒫(B)
⊢ ∀a∈A.|f(a)|=1
b∈f(a)
Parametrized Iota-Conversion
b=f(a)

Examples¶

We call a finite automaton with transition function δ:Q×Σ→𝒫(Q) deterministic if ⊢∀(q,c)∈Q×Σ.|δ(q,c)|=1 holds. In this case we can regard δ as a function from Q×Σ to Q which allows us (by abuse of notation) to write expressions of the form q’∈δ(q,c) as q’=δ(q,c) which is captured by the following instance of the above rewrite rule:

⊢ δ:Q×Σ→𝒫(Q)
⊢ ∀(q,c)∈Q×Σ.|δ(q,c)|=1
q′∈δ(q,c)
Parametrized Iota-Conversion
q′=δ(q,c)
