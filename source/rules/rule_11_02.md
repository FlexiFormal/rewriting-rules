Rule 11.2: "Has"-NP Double Negation Elimination
===============================================


```{rewrite-rule}
$\vdash \forall \varphi.\, \neg \neg \varphi = \varphi$

$x$ does not have no {ph}`NP`

$x$ has a {ph}`NP`
```


Examples
--------

```{rewrite-rule}
$x$ does not have no element

$x$ has an element
```
