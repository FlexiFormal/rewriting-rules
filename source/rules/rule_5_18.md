Rule 5.18: Relation Guard Reduction
===================================


```{rewrite-rule}
$\{R[y] \mid \varphi[y_1,\dots,y_n]\}$

$\{y \mid R[y] \text{ and } \varphi[y]\}$
```


Examples
--------

```{rewrite-rule}
$\{i \leq i \leq n \mid 1/i < \varepsilon\}$

$\{i \mid i \leq i \leq n \text{ and } 1/i < \varepsilon\}$
```
