Rule 4.9: "Or"-NP Aggregation
=============================


```{rewrite-rule}
$x$ is $d_1$@(a | no) {ph}`NP`$_1$ or ... or $x$ is $d_n$@(a | no) {ph}`NP`$_n$

$x$ is $d_1$ {ph}`NP`$_1$ or ... or $d_n$ {ph}`NP`$_n$
```

Examples
--------

```{rewrite-rule}
$x$ is a proper class or $x$ is a set

$x$ a proper class or a set
```
