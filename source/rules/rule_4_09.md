Rule 4.9: "Or"-NP Aggregation
=============================


```{rewrite-rule}
$x$ is (a | no) {ph}`NP`$_1$ or ... or $x$ is (a | no) {ph}`NP`$_n$

$x$ is (a | no) {ph}`NP`$_1$ or ... or (a | no) {ph}`NP`$_n$
```

Examples
--------

```{rewrite-rule}
$x$ is a proper class or $x$ is a set

$x$ a proper class or a set
```
