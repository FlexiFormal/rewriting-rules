Rule 11.1: "Is"-NP Double Negation Elimination
==============================================


```{rewrite-rule}
NP : <NP> = "prime number"

P : <Var> = "$P$"

x : <Term> = "$x$"

|- $\forall :P:.\, \neg \neg :P: = :P:$

":x: is not no :NP:"

--- "Is"-NP Double Negation Elimination

":x: is a :NP:"
```


Examples
--------

```{rewrite-rule}
|- $\forall P.\, \neg \neg P = P$

"$x$ is not no prime number"

--- "Is"-NP Double Negation Elimination

"$x$ is a prime number"
```
